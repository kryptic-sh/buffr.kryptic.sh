buffr

Vim-inspired browser. Native, GPU-accelerated. Rust + CEF.

Status: early. Nothing to install yet. Watch the repo for releases.

why

  • Modal browsing. Hint-mode for links, ex-style command palette.
  • Native window, native fonts. No Electron, no JVM, no webview wrapper.
  • Rust + CEF — Chromium rendering without an Electron-shaped layer above it.
  • Mouse + keyboard, both, always.
